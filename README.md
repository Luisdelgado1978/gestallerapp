# gestallerapp- unidad 3
aplicacion desarrllada en " ionic React"
como parte del trabajo de la unidad 3 del 
curso de desarrollo de aplicaciones.
el proyecto incluye navegacion con Tabs, 
un conjunto de paginas funcionales y una 
estructura compatible para ejecucion local usando Ionic.

## Instalacion y ejecucion
### 1.clonar el repositorio
'''  bash
git clone
https://github.com/luisdelgado1978/gestallerapp.git
cd gestallerapp
'''

### 2. Instalar dependencias
''' bash
npm install
''''

### 3. Ejecutar la aplicacion
'''bash
ionic serve
'''
esto abrira la app en:
http://localhost:8100/
'''
